## Objetivo do Trabalho

Este MVP tem como objetivo analisar os fatores que influenciam a rotatividade
(Attrition) de funcionários em uma organização, utilizando um conjunto de dados
público de recursos humanos da IBM.

O problema de negócio abordado é a dificuldade das empresas em compreender
quais características dos funcionários estão associadas à sua saída.

As perguntas de negócio que orientam a análise são:
1. Quais fatores demográficos estão associados à rotatividade?
2. A renda mensal e a satisfação no trabalho influenciam a saída?
3. Existem cargos ou departamentos com maior incidência de Attrition?
4. Quais padrões podem auxiliar na redução da rotatividade?
